Accéder à Relevé d’emploi sur le Web (RE Web)

Rappels importants :

  • Si vous êtes incapable d’ouvrir une session à l’aide de votre CléGC, vous pouvez quand même récupérer l’accès à votre compte RE Web.
  • Si vous êtes déjà inscrit à RE Web, vous n’avez pas besoin d’inscrire une nouvelle organisation. Une fois que votre session est ouverte, vous n’avez qu’à cliquer sur le nom de votre organisation pour accéder à vos numéros d’entreprise.

Vous devez ouvrir une session pour vous inscrire à RE Web en ligne ou accéder à votre compte.

Choisissez une méthode d’ouverture de session sécurisée :

Utilisez un Partenaire de connexion

Sélection d’un Partenaire de connexion

  • Saisissez les mêmes données d’ouverture de session que celles que vous utilisez pour d’autres services en ligne (p. ex., services bancaires en ligne).
  • Aucun de vos renseignements (p. ex., financiers, bancaires) ne sera divulgué à EDSC/Service Canada. Votre Partenaire de connexion ne saura pas quel service du gouvernement vous utilisez.
  • Vous quitterez temporairement le site Web de EDSC/Service Canada afin d’utiliser votre Partenaire de connexion.

Ouvrez une session ou enregistrez-vous pour obtenir un ID d’utilisateur CléGC et un mot de passe :

Poursuivez vers la CléGC

  • Ouvrez une session à l’aide de votre ID d’utilisateur CléGC et de votre mot de passe si vous n’utilisez pas l’un des Partenaires de connexion.
  • Enregistrez-vous afin d’obtenir un ID d’utilisateur CléGC et un mot de passe si vous n’en avez pas.

Récupérer l’accès à votre compte RE Web

Afin de récupérer l’accès à votre compte RE Web, vous aurez besoin de votre numéro de référence de l’utilisateur. Vous devrez également répondre à vos questions de sécurité.

  1. Choisissez votre méthode d’ouverture de session : partenaire de connexion ou CléGC.
    • Si vous utilisez un partenaire de connexion, choisissez votre institution financière et entrez les renseignements que vous utilisez pour vous connecter à vos services bancaires en ligne.
    • Si vous choisissez de créer une nouvelle CléGC, cliquez sur le bouton Poursuivez vers la CléGC. Une fois à la page de bienvenue de la CléGC, cliquer sur le bouton S’enregistrer à la gauche de l’écran. Prenez note de votre nouveau nom d’utilisateur et votre nouveau mot de passe et gardez-les en lieu sûr.
  2. Une fois que vous avez accepté l’énoncé de confidentialité et les modalités d’utilisation de RE Web :
    • Vous serez redirigé vers la page « Créer mon profil », où vous pourrez vous inscrire comme nouvel utilisateur ou récupérer l’accès à votre profil.
    • Sélectionner l’option 2 : « Récupérer accès à mon profil Relevé d'emploi sur le Web existant en donnant mon numéro de référence d'utilisateur et en répondant à mes questions de sécurité » ou cliquer sur le lien Retrouver mon numéro de référence d’utilisateur
  3. Si vous avez choisi l’option 2 :
    • Inscrivez votre numéro de référence de l’utilisateur, puis cliquez sur Continuer.
    • Répondez à vos trois questions de sécurité et cliquez sur Continuer.
    • Dans la page de confirmation, cliquez sur Continuer pour commencer à utiliser votre nouveau nom d’utilisateur et votre nouveau mot de passe pour accéder à votre compte RE Web.
  4. Si vous cliquez sur le lien Retrouver mon numéro de référence d’utilisateur
    • Inscrivez votre Adresse courriel associée et cliquer sur Continuer
    • Répondez à vos trois questions de sécurité et cliquez sur Continuer.
    • Dans la page de confirmation, cliquez sur Continuer pour commencer à utiliser votre nouveau nom d’utilisateur et votre nouveau mot de passe pour accéder à votre compte RE Web.
